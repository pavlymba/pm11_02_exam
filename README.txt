ПРОЕКТ: КНИЖНЫЙ МИР
База данных для автоматизации книжного магазина

СОСТАВ ПРОЕКТА:
1. SQL_скрипт_КнижныйМир.sql - полный скрипт создания БД с тестовыми данными
2. ER_диаграмма_КнижныйМир.pdf - схема базы данных

ОСОБЕННОСТИ:
- Нормализованная структура (3НФ)
- Ролевая модель: Гость, Клиент, Менеджер, Администратор
- Полная система заказов с пунктами выдачи
- Каталог книг с авторами, издательствами и жанрами

ТЕСТОВЫЕ ДАННЫЕ:
- 4 роли пользователей
- 9 тестовых пользователей
- 15 авторов, 8 жанров, 6 издательств
- 5 книг с полной информацией
- 2 завершенных заказа

Для развертывания выполните SQL-скрипт в любой СУБД MySQL.
